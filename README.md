# Mean-Variance-Standard Deviation Calculator

This project is part of the **freeCodeCamp – Data Analysis with Python** certification.

## Description
A Python program that uses **NumPy** to calculate the mean, variance, standard deviation, maximum, minimum, and sum of a 3×3 matrix.

The function accepts a list of nine numbers, converts it into a NumPy array, and returns the results in a dictionary format.

## Technologies Used
- Python
- NumPy

## Files
- `mean_var_std.py` – contains the `calculate()` function
- `main.py` – used for testing
- `README.md` – project documentation

## Example Usage
```python
from mean_var_std import calculate

calculate([0,1,2,3,4,5,6,7,8])
